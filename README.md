# 🍕 Teknolojik Yemekler - Responsive Food Delivery UI

A modern, fully responsive landing page designed for a food delivery platform. Built with Vite and Vanilla JavaScript, this project showcases advanced CSS architecture, mobile-first design principles, and smooth user interactions.

## 🧠 The "Why" behind the Project
Creating a frictionless user journey is critical in e-commerce. With a strong focus on user-centric design, this interface utilizes clear visual hierarchies, accessible layouts, and intuitive navigation. By applying principles of cognitive ease, the layout guides users naturally from mouth-watering promotional banners directly to the call-to-action buttons.

## ✨ Key Features
* **📱 Fully Responsive Design:** Adapts seamlessly to mobile and tablet devices using custom CSS media queries (`max-width: 768px`), completely restructuring the layout for smaller screens.
* **🎨 Advanced CSS Layouts:** Extensive use of **Flexbox** for alignment and **CSS Grid** for complex gallery sections (e.g., footer image grids).
* **✨ Interactive UI/UX:** Smooth CSS transitions and hover effects (`transform: scale`) on navigation links, order buttons, and product cards to provide immediate visual feedback.
* **⚡ Lightning Fast:** Powered by **Vite** for a highly optimized development experience and rapid asset loading.

## 🛠️ Tech Stack
* **Build Tool:** Vite
* **Logic:** Vanilla JavaScript
* **Styling:** Pure CSS3 (Flexbox, CSS Grid, Media Queries, Custom Fonts)
